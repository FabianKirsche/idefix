pre-requisite

  1. Powershell version > 3.0
    Download Powershell 3.0
  2. open a command window e.g. via Windows + R and cmd
  3. create a directory for the installation (avoid spaces in directory path) e.g. mkdir c:\eclipseInstaller
  4. enter newly created folder e.g. cd /d c:\eclipseInstaller
  5. proceed with the installation sections

Problems/FAQ

  • Problem: Installer fails during download

    Reason: network problems due to e.g. time-out, proxies, ssl-accelerator, ...

    Solution:try "Back" and "Next" once or twice. It will try to reload. If error persists contact admin.

  • Problem: Exception calling "DownloadFile" with "2" argument(s): "The remote server returned an error: (407) Proxy Authentication Required."

    Solution: open InternetExplorer and goto a Internet Site (not Intranet). After site is loaded do the cmd execution again.

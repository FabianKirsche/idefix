pre-requisite

  1. open a terminal
  2. create a directory for the installation e.g. mkdir ~/idefix
  3. enter newly created folder e.g. cd ~/idefix
  4. proceed with the installation sections

installation

  1. execute the following line inside terminal in the folder you want to install
  2. installation - downloads, and installs the EclipseInstaller into the directory where the call is made and launches it in Advanced Mode

eclipseInstaller - oomph

  1. select from the section IDEfix klib.io the product you want to install via double-click
  2. scroll down to Projects of klib.io and select CEC 16.1 via double-click. It must appear inside the table at the bottom of the window. -> Next
  3. Next
  4. Finish
  5. Press "Accept" in the appearing "Licenses" pop-up window
  6. IDEfix Window should appear with the "Bndtools - Eclipse" perspective open.
  7. eclipseInstaller can be closed

Startup of IDEfix installations

  • IDEfix can be started from now via <installDir>\Eclipse Installer.app

clean-up

  1. execute the following line inside the cmd window in the folder you have the installation
  2. cleans all locally used directories (only bundle pool remains)

Execute the following inside a cmd window

  1. downloads, and installs the EclipseInstaller into the directory where the call is made and launches it in Simple Mode
    cls & powershell -command "& {(New-Object System.Net.WebClient).DownloadFile('http://peterkir.github.io/idefix/bootstrap/win32/engage_simple.cmd','engage_simple.cmd')}" & engage_simple.cmd
  2. downloads, and installs the EclipseInstaller into the directory where the call is made and launches it in Advanced Mode
    cls & powershell -command "& {(New-Object System.Net.WebClient).DownloadFile('http://peterkir.github.io/idefix/bootstrap/win32/engage.cmd','engage.cmd')}" & engage.cmd
  3. cleans all locally used directories (only bunlde pool remains)
    cls & powershell -command "& {(New-Object System.Net.WebClient).DownloadFile('http://peterkir.github.io/idefix/bootstrap/win32/clean.cmd','clean.cmd')}" & clean.cmd

FAQ

  • Problem: Exception calling "DownloadFile" with "2" argument(s): "The remote server returned an error: (407) Proxy Authentication Required."

    Solution: open InternetExplorer and goto a Internet Site (not Intranet). After site is loaded do the cmd execution again.
